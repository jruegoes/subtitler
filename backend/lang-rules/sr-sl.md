# Serbian → Slovenian Translation Rules

These are specific rules for translating Serbian subtitles into Slovenian.

## False Friends (CRITICAL — reverse direction!)

| Serbian | Serbian meaning | Slovenian equivalent | Slovenian false friend meaning |
|---------|-----------------|---------------------|-------------------------------|
| `grad` | city/town | `mesto` | `grad` in Slovenian = castle |
| `mesto` | place/spot | `kraj` / `prostor` | `mesto` in Slovenian = city |
| `trudna` | pregnant | `noseča` | `trudna` in Slovenian = tired (archaic) |
| `časopis` | magazine | `revija` | `časopis` in Slovenian = newspaper |
| `pozorište` | theater | `gledališče` | — |
| `hleb` | bread | `kruh` | — |
| `dete` | child | `otrok` | — |
| `lek` | medicine | `zdravilo` | — |
| `avion` | airplane | `letalo` | — |
| `računar` | computer | `računalnik` | — |
| `kasa` | cash register | `blagajna` | — |
| `karta` | ticket | `vozovnica` (transport) / `vstopnica` (event) | — |
| `sud` | court | `sodišče` | — |
| `slika` | picture | `slika` (same) | — |

## Plural → Dual Conversion
Serbian has no dual form but Slovenian DOES. When context clearly involves exactly TWO people/things, use Slovenian dual:
- `mi smo išli` (we went) → `midva sva šla` (if 2 men) / `midve sva šli` (if 2 women) — when context is clear
- `vi ste rekli` (you said) → `vidva sta rekla` (if addressing 2 people)
- `oni su` (they are) → `onadva sta` (if exactly 2)

Only use dual when the context CLEARLY indicates exactly two entities. When unsure, use plural.

## Turkish/French Loanwords → Slovenian Equivalents
Serbian words of Turkish/French origin should be translated to standard Slovenian:
- `komšija` (SR, neighbor) → `sosed` (SL)
- `budala` (SR, fool) → `norec` / `bedak` (SL)
- `papuča` (SR, slipper) → `copat` (SL)
- `šofer` (SR, driver) → `voznik` (SL)
- `garderoba` (SR, wardrobe) → `garderoba` (SL, same word used)
- `čaršija` (SR, bazaar) → `tržnica` / `čaršija` (SL, depends on context)
- `ćevap` → `čevap` (SL, adapted spelling)
- `ajde` (SR, come on) → `daj no` / `gremo` (SL)

## Ekavian → Slovenian
Serbian Ekavian forms have no direct mapping — translate to the Slovenian word:
- `mleko` (SR) → `mleko` (SL, same)
- `pesma` (SR) → `pesem` (SL)
- `dete` (SR) → `otrok` (SL)
- `reka` (SR) → `reka` (SL, same)
- `vreme` (SR) → `čas` (time) / `vreme` (weather) — CAREFUL: context-dependent

## Grammar Adjustments
1. **Remove vocative**: Slovenian has no vocative case. Serbian vocative must become nominative.
   - `Marko!` → `Marko!` (names stay same)
   - `druže!` → `prijatelj!` (friend, nominative)
   - `brate!` → `brat!` (brother, nominative)
2. **Infinitive vs da+present**: Serbian `da` + present often maps to Slovenian infinitive.
   - `Hoću da radim` (SR) → `Hočem delati` (SL)
3. **Future tense**: Serbian contracted `radiću` → Slovenian `bom delal/delala`
4. **Reflexive pronoun placement**: Different in Slovenian — `se` usually goes after the verb.

## Register and Tone
- Serbian speech is typically more direct and expressive
- Slovenian speech is softer and more reserved
- Tone down overly direct Serbian expressions to sound natural in Slovenian
- Serbian `Ma daj!` → `Daj no!` or `No daj!` (SL)
- Serbian `Šta je?!` → `Kaj je?!` (SL, if using colloquial) or `Kaj se dogaja?` (SL)

## Vikanje/Tikanje
Match the Serbian formality level:
- Serbian `ti` → Slovenian `ti`
- Serbian `Vi` (formal) → Slovenian `Vi` (capital V for respect)
- Serbian `-te` verb endings (formal) → Slovenian formal conjugations

## Idiomatic Expressions
- `Šta ja znam` (SR) → `Kaj pa jaz vem` (SL)
- `Ma daj` (SR) → `Daj no` (SL)
- `Ajde` (SR) → `Gremo` / `Daj` (SL)
- `Važi` (SR) → `Velja` / `OK` (SL)
- `Kapiram` (SR) → `Razumem` / `Kapiram` (SL, also used colloquially)
- `Jel da?` (SR) → `A ne?` / `Kajne?` (SL)
- `Znaš šta` (SR) → `A veš kaj` (SL)
