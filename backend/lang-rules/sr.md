# Serbian Target Language Rules

You are translating INTO Serbian. Follow these Serbian-specific conventions:

## Script
Use **Latin script** for subtitles (not Cyrillic). Serbian Latin uses these special characters: č, ć, š, ž, đ. Never substitute with plain ASCII (c, s, z, d).

## Dialect
Use **Ekavian** dialect (standard Serbian, Belgrade standard):
- `mleko` NOT `mlijeko` (milk)
- `pesma` NOT `pjesma` (song)
- `dete` NOT `dijete` (child)
- `reka` NOT `rijeka` (river)
- `vreme` NOT `vrijeme` (time/weather)
- `vera` NOT `vjera` (faith)
- `mesto` NOT `mjesto` (place)

## Vocative Case
Serbian has a vocative case that Slovenian does NOT have. Use it for direct address:
- `Marko` → `Marko!` (vocative same)
- `Jure` → `Jure!`
- `druže` (friend, vocative of `drug`)
- `brate` (brother, vocative of `brat`)
- `čoveče` (man, vocative of `čovek`)

## Formal/Informal Address
- `ti` = informal, verb ending changes (e.g., `Izvini` = sorry informal)
- `Vi` = formal, verb ending `-te` (e.g., `Izvinite` = sorry formal)
- Match the formality level from the Slovenian source

## Future Tense Contractions
When infinitive ending in `-ti` is followed by auxiliary `ću/ćeš/će/ćemo/ćete/će`, they combine:
- `radiću` = I will work (NOT `raditi ću`)
- `reći ću` = I will say (stays separate when infinitive doesn't end in `-ti`)

## Numbers
- Write out numbers 1–10: `jedan`, `dva`, `tri`, `četiri`, `pet`, `šest`, `sedam`, `osam`, `devet`, `deset`
- Use numerals for 11+
- Decimal separator: comma (`23,99`)
- Date format: `20.2.2004.` (period after year too)
- Use 24-hour time format

## Punctuation
- Ellipsis: use `…` (Unicode character), NOT three periods
- Quotation marks: `„ "` (opening low, closing high)
- No periods in acronyms: `BBC`, `CIA`, `EU`
- Speaker change: hyphen at start of line, no space after: `-Šta hoćeš?`

## Common Serbian Expressions
Use natural Serbian phrasing, not literal translations:
- "Nema problema" = no problem
- "Ma daj" = come on / no way
- "Šta ima?" = what's up?
- "Ajde" = come on / let's go
- "Važi" = OK / deal
- "Kapiram" = I understand (informal)
- "Jel da?" = right? / isn't it?

## Turkish/French Loanwords in Serbian
Serbian naturally uses these — prefer them over literal translations when they sound more natural:
- `komšija` (Turkish) = neighbor (NOT `sused`)
- `čaršija` = bazaar/market area
- `ćevap` = grilled meat dish
- `budala` (Turkish) = fool
- `šofer` (French chauffeur) = driver
- `parter` (French parterre) = ground floor
- `garderoba` (French) = wardrobe/cloakroom

## Name vs. Word Disambiguation

These Serbian/Croatian/Bosnian/Montenegrin words MUST be translated, NOT kept as names:
Ali=But, Ili=Or, Da=Yes/That, Ja=I, Je=Is, Sam=Alone/I am, Ti=You, Mi=We, Ne=No, Pa=So/Well, Kad=When, Jer=Because, Ma=But, Sad=Now, Dok=While, To=That, Ni=Nor, Već=Already, Zar=Really?, Ima=Has

When capitalized at the start of a sentence, these are still common words — translate them. Only keep ACTUAL person names untranslated.
