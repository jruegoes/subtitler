# Slovenian Source Language Rules

You are translating FROM Slovenian. Be aware of these Slovenian-specific features:

## Dual Form (Dvojina)
Slovenian has THREE grammatical numbers: singular, dual, plural. Most target languages only have singular/plural.
- `Greva` = let's go (2 people) vs `Gremo` = let's go (3+)
- `midva/midve` = we two (masc/fem) vs `mi` = we (3+)
- `vidva/vidve` = you two vs `vi` = you (3+)
- `onadva/onidve` = they two vs `oni/one` = they (3+)
- Dual verbs: `-va` ending (e.g., `delava` = we two work, `delamo` = we work)

When translating dual forms, use plural in the target language BUT preserve the intimacy/closeness the dual implies — it often signals a close relationship between exactly two people.

## Vikanje/Tikanje (Formal/Informal)
- `ti` = informal you → match the formality level in the target language
- `Vi` (capital) = formal you → use formal register in the target language
- Context clues: family/friends use `ti`, strangers/business use `Vi`

## Common Slovenian Particles
These short words carry important meaning — do NOT skip them:
- `pač` = well/simply/just (softener, no direct translation — convey the casual tone)
- `no` = well/so (conversation filler)
- `a ne` = right?/isn't it? (tag question)
- `baje/baje(že)` = supposedly/apparently
- `itak` = anyway/obviously

## Punctuation
- Slovenian uses `„ "` or `» «` quotation marks
- Comma before subordinate clauses (`da`, `ki`, `ko`, `če`, `ker`)
- Decimal: comma (`23,99`), thousands: period or space

## Germanisms in Slovenian
Slovenian has many German loanwords that should be translated to the target language's natural equivalent, NOT transliterated:
- `šnita` (Schnitte) = slice
- `štrudel` = strudel
- `šparati` (sparen) = to save
- `rajža` (Reise) = trip/journey
- `šlapa` (Schlappe) = slipper
- `cajt` (Zeit) = time (colloquial)
- `fajn` (fein) = nice/fine

## Name vs. Word Disambiguation

These Slovenian words MUST be translated, NOT kept as names:
Kaj=What, Ali=But/Or, To=This/That, Ko=When, Pa=And/But, Da=That/Yes, Sem=I am, Ti=You, Mi=We, Ne=No, Ja=Yes/I, Le=Only, Že=Already, Še=Still, Ma=My, Bi=Would, Bo=Will, Ana=interrogative particle, Dan=Day, Dar=Gift

When capitalized at the start of a sentence, these are still common words — translate them. Only keep ACTUAL person names untranslated.
